Recovery Drones - pacote pronto para hospedagem

Arquivos incluídos:
- index.html
- robots.txt
- sitemap.xml
- 50 páginas locais de cidades do RS

Para publicar:
1. Extraia o ZIP.
2. Envie todos os arquivos para a raiz da hospedagem ou do GitHub Pages.
3. Ajuste o domínio final no index.html, sitemap.xml e páginas locais se necessário.
