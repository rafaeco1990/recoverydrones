# Recovery Drones RS | Pacote Dominar Total

## O que vem neste ZIP
- Home renovada
- 75 páginas de cidades em `/cidades`
- 6 páginas por problema em `/problemas`
- 450 páginas do tipo cidade + problema em `/estrategias`
- `cobertura-rs.html` para concentrar links locais sem poluir o menu
- `sitemap.xml` e `robots.txt`

## Como aplicar
1. Extraia o ZIP.
2. Envie todos os arquivos para o repositório do GitHub Pages.
3. Confirme que as pastas `/cidades`, `/problemas`, `/estrategias` e `/assets` foram publicadas.
4. Acesse `https://seu-dominio/sitemap.xml` e confira se abriu.
5. No Search Console, reenvie o sitemap e solicite indexação das páginas principais.

## Sobre as cidades ocultas
As páginas locais ficam fora do menu principal. Elas continuam indexáveis via sitemap e links internos.
