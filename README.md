# Recovery Drones - pacote pronto para GitHub Pages

## Publicação rápida
1. Envie todos os arquivos desta pasta para o repositório do GitHub.
2. Em Settings > Pages, selecione a branch principal e a pasta `/root`.
3. Aguarde a publicação.

## Arquivos principais
- `index.html`: home
- `style.css`: estilos
- `script.js`: interações
- `robots.txt`
- `sitemap.xml`
- `*.html`: 225 páginas locais
